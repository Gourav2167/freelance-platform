# Vasudha: Next-Gen Freelance Market

## Project Overview
Vasudha is an immersive, high-performance freelance ecosystem designed to bridge the gap between global talent and organizations. By leveraging cutting-edge web technologies and a refined 3D interface, it provides a professional environment for project search, bidding, and contract management.

## Key Features
- **Dynamic Project Market**: Real-time project discovery via an interactive 3D constellation interface.
- **Secure Bidding System**: Seamless bid submission and tracking for freelancers.
- **Organizational Dashboard**: Tools for organizations to post projects, manage bids, and hire talent.
- **Integrated AI Assistant**: Professional AI guidance for project analysis and directory navigation.
- **Professional Terminology**: Localized for Indian professional markets with ₹ (Rupee) currency support.

## Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **3D Engine**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) / [Three.js](https://threejs.org/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Backend/Database**: [Supabase](https://supabase.com/) (Auth, PostgreSQL, Realtime)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Notifications**: [Sonner](https://sonner.emilkowal.ski/)
